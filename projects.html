Project 1: Doctor Appointment Management System

The Doctor Appointment Management System streamlines appointment scheduling, allowing patients to book online. It improves efficiency, patient satisfaction, and time management for both patients and doctors, enhancing the overall healthcare experience.

Project 2: Focus Pause

I created a Chrome extension that automatically pauses YouTube videos when switching between Chrome tabs and windows, making it easier to focus on work or learning without the hassle of repeatedly pausing the video.

Project 3: WiFi QR Code Generator

This is a simple HTML, CSS, and JavaScript project that lets you generate a WiFi QR Code by just entering the WiFi name and password. Once generated, the QR code can be downloaded as an image or a PDF, making it easy to print or share for quick WiFi access without typing credentials.
